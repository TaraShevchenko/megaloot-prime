# Monsters

![[Monster.png]]

## Статы и HP
- У монстра есть набор статов.
- Из статов формируется `HP` (как отдельное состояние).

## Бой
- Каждая атака игрока уменьшает `HP`.
- Когда `HP` заканчивается — монстр умирает.

## Дроп
- После смерти монстр выдаёт дроп.
- Каждый предмет дропа имеет шанс выпадения.
