# Equipment inventory

![[Equipment inventory.png]]![[Equipment inventory (full).png]]

## Назначение
Основной инвентарь экипировки персонажа.

## Правила слотов
- В каждом слоте может лежать только предмет экипировки (любой тип).
- Предметы в этом инвентаре считаются предметами персонажа: они доступны для экипировки и улучшения.
