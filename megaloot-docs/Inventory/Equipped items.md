## Правила
- Каждый слот может содержать только строго тот тип экипировки, за которым закреплён слот.
- При перемещении предмета на слот:
  - если тип подходит — предмет экипируется;
  - если тип не подходит — игрок получает всплывающее уведомление об ошибке.
- Если слот уже был экипирован — экипировка меняется местами (swap).
