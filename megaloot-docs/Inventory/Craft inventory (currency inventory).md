## Назначение
Инвентарь для применения к предмету экипировки валют/крафтовых ресурсов, с отображением таблицы статов для удобного выбора и контроля изменений.

## Макет
- 4 колонки валют (по 2 с каждой стороны).
- Центральный слот: предмет экипировки (любой тип).
- Таблица характеристик (статов) выбранного предмета.

## Валюты и эффекты
### Колонка 1 — красные гемы (уровни 1–6)
- Каждый гем добавляет соответствующую характеристику в предмет (уровень гема задаётся позицией 1–6).

### Колонка 2 — «магические капли» и риск
1. Красная маг. капля — удаляет случайную характеристику у вещи.
2. Жёлтая маг. капля — удаляет выбранную характеристику.
3. Зелёная маг. капля — улучшает значение случайной характеристики (возможно выше диапазона; ориентир: +10–20%). Одна и та же характеристика может быть улучшена этим типом эффекта только один раз (и аналогичными предметами).
4. Синяя маг. капля — улучшает выбранную характеристику по правилам зелёной капли.
5. Сфера удачи — пытается улучшить 2 случайные характеристики по тем же правилам, но с шансом 50% вместо улучшения удаляет одну характеристику и ничего не улучшает.
6. Клонирование — клонирует предмет.

### Колонка 3 — ресурсы (золото/пыль/слитки)
1. Золото — используется для покупки вещей или валют.
2. Пыль — добывается распылением предметов; используется для улучшения других валют, например:
   - красные гемы (добавление характеристик) → следующий уровень;
   - магические капли: красная → жёлтая, зелёная → синяя;
   - слитки → следующий уровень;
   - сферы/гемы реролла → следующий уровень.
3. Слиток повышения редкости `Uncommon`.
4. Слиток повышения редкости `Rare`.
5. Слиток повышения редкости `Epic`.
6. Слиток повышения редкости `Legendary`.

### Колонка 4 — синие гемы реролла
- Гемы рероллят значение соответствующей характеристики; уровень (позиция 1–6) задаёт «силу/уровень» реролла.

## Взаимодействие
- ПКМ по валюте — контекстное меню.
- ЛКМ + перетаскивание валюты на предмет — применяет эффект/улучшение.
- Чтобы положить предмет в центральный слот — перетащить его из любого другого инвентаря.
- Если перетащить валюту на предмет не в craft inventory, валюта всё равно применяется; отличие craft inventory — наличие таблицы характеристик, которая упрощает процесс.

## Ограничения слотов
- Каждый слот craft inventory принимает только «свою» валюту (фиксированный тип в каждом слоте).
- Центральный слот рассчитан исключительно под предмет экипировки (любой тип).

## Заметки/предложения
- Валюты, которые не применяются к предмету напрямую (например, золото и пыль), логичнее держать в менее приоритетных слотах, а слитки — поднять выше.
